Hubungkan akun Google

Export semua transaksi ke spreadsheet pribadimu lewat OAuth. Datamu tetap di browser dan dikirim langsung ke Google — tidak lewat server mana pun.

  1. Buka Google Cloud Console, buat project, lalu aktifkan Google Sheets API.
  2. Buat OAuth Client ID tipe Web application.
  3. Di Authorized JavaScript origins, tambahkan domain situs ini (mis. origin).
  4. Salin Client ID lalu tempel di kolom bawah.
